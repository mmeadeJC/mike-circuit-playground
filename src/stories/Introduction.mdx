# Circuit Design System Playground

Welcome to the **Circuit Design System** playground. This Storybook instance serves as a development and prototyping environment for the JumpCloud Circuit Design System.

## Navigation

### Circuit DS / Components

All available design system components in one place:

- **PrimeVue Primitives** -- Avatar, Button, Checkbox, DataTable, Dialog, Divider, Drawer, InputText, Message, Panel, RadioButton, Select, SelectButton, Skeleton, Tag, Textarea, TieredMenu, Toast, ToggleSwitch, Tooltip, and more
- **Custom Components** -- CardButton, CheckboxWithLabel, Chip, CollapsiblePanel, Dropdown, FormField, InputOtp, MessageNotification, Paginator, Password, ProgressSpinner, RadioButtonWithLabel, SelectWithSlots, ToastNotification, AppNavigation, TourGuide components

Each component is styled using Circuit's Tailwind preset and PrimeVue passthrough theme configuration.

### Projects / Asset Management

A prototyping space for the Asset Management feature:

- **Mock Data** -- Realistic sample data for devices, accessories, and locations
- **Pages** -- Empty page canvases for experimenting with layouts (Device List, Accessory List, Locations, Device Detail)

## Tech Stack

- **Vue 3** -- Composition API
- **Tailwind CSS 4** -- Utility-first CSS with Circuit's design tokens
- **PrimeVue 4** -- Unstyled mode with Circuit passthrough templates
- **Storybook 8** -- Component development environment
- **@jumpcloud/circuit** -- Design system package
